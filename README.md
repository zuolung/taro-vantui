> 一套适用于Taro3, 基于react开发的vantui组件库

## 为什么需要

基于Vant-weapp改造的Taro组件库, 支持h5、weapp、alipay

## 安装

```bash
yarn add taro-vantui
```

## 使用

src/style/index.less

```css
@import 'node_modules/taro-vantui/dist/style/index';
```

```js
import { Button } from 'taro-vantui'
```
